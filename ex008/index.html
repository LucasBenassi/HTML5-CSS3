Principais Formatações

Negrito / Destaque

Nesta frase, temos um termo em Negrito usando a Tag 'B'.

Nesta frase, tenho um termo em destaque utilizando a tag 'strong'.

Itálico / Ênfase

Nesta frase, temos um termo destacado em itálico utilizando a tag 'i'.

Nesta frase, temos um termo em ênfase usando a tag 'em' (semântica).

Texto Marcado

Podemos criar, também, um texto marcado usando a tag 'mark'.

Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo.

Texto Deletado

Podemos marcar um texto como Excluído para indicar que ele deve ser lido, mas não considerado.

Texto Inserido

Podemos marcar um texto como Inserido para dar uma ênfase e indicar que ele foi adicionado posteriormente.

Existe, também, a tag 'u', que não é semântica.

Texto Sobrescrito

Para inserir coisas do tipo X2+3

Texto Subscrito

Para escrever coisas como, por exemplo, H2O.
